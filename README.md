> Gestione turni, spettanze e straordinari per i lavoratori a turni

- Calendario turni con calcolo automatico di CO, FS, RC, RR
- Scanner OCR per importare turni da foglio carta
- Calcolo FESI, indennità e straordinario
- Export PDF, sincronizzazione Calendario iPhone
- Gratuita · Acquisto opzionale per rimuovere la pubblicità
